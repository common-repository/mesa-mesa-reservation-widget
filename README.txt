Reservation widget for Mesa Mesa restaurant software

== Installation ==

1. Upload `mesa-mesa-reservation-widget/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your Guestserve ID in the settings page.
1. Place `[mesa_mesa_widget]` in your templates. See futher customization options on the settings page.
